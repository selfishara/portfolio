# Maps App

## Overview

Maps App is an Android application that integrates map-based functionality, allowing users to interact with geographical data through a mobile interface.

The project focuses on implementing map visualization, marker interaction, and user interface elements within a modern Android architecture.

---

## Technologies Used

- Kotlin
- Jetpack Compose
- Android Studio
- Maps SDK
- REST APIs

---

## Key Features

- Interactive map interface
- Marker creation and interaction
- Map-based UI components
- Responsive mobile design

---

## My Role

In this project I worked on:

- UI development with Jetpack Compose
- Map integration and marker handling
- Screen layout design
- Interaction logic for map elements
