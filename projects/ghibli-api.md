# Studio Ghibli API Project

## Overview

This project is an application built to consume and display data from the public Studio Ghibli API.

The goal of the project was to practice working with REST APIs, handling JSON data, and integrating external services into an application.

---

## Technologies Used

- Kotlin
- REST APIs
- Retrofit
- JSON parsing

---

## Key Features

- Fetching data from an external API
- Displaying structured information from API responses
- Handling asynchronous network calls
- Basic UI integration with API data

---

## My Role

In this project I worked on:

- API integration
- Handling network requests
- Parsing JSON responses
- Displaying API data in the application interface
