# Gym BDOR

## Overview

Gym BDOR is a database-focused project designed to model and manage a gym management system.

The project explores relational and object-relational database concepts, including data modeling, table relationships, and structured queries.

---

## Technologies Used

- SQL
- PostgreSQL
- Database modeling
- Relational schema design

---

## Key Features

- Structured database schema
- Relationship modeling between entities
- Query-based data retrieval
- Database design for a real-world scenario
